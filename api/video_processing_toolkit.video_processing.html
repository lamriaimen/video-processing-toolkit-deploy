video_processing_toolkit.video_processing.get_frame_types(video_file)[source]

Function to extract the type of each frame in a video using ffprobe. The frame types are typically: - ‘I’ (Intra-coded): A full image - ‘P’(Predicted): Stores only the difference from the previous frame - ‘B’ (Bidirectional): Stores differences using both previous and next frames.

Parameters:

video_file (str) – Path to the input video file.

Returns:

A zip object containing tuples of (frame_index, frame_type),

where frame_index is an integer and frame_type is a string (‘I’, ‘P’, ‘B’).

Return type:

Zip
