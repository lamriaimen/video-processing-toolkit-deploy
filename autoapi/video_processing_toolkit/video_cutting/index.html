video_processing_toolkit.video_cutting.get_clip(input_filename, output_filename, start_time, end_time)[source]

Cuts a video using ffmpeg while preserving the original framerate and forcing one I-frame per second for clean frame extraction.

Parameters:

  • input_filename (str) – Path to the input video.

  • output_filename (str) – Path to save the output video.

  • start_time (float) – Start time in seconds.

  • end_time (float) – End time in seconds.

video_processing_toolkit.video_cutting.cut_video_clips(input_filename, output_file_path, time_stamp_start_1, time_stamp_end_1)[source]

Function to read several start and end time stamps, mentioned in two vectors; “time_stamp_start_1” and “time_stamp_end_1”, to crop the video clips based on these start and end time stamps.

Parameters:

  • input_filename (str) – Path of the input video file

  • output_file_path (str) – The path of the output file clip

  • time_stamp_start_1 (List[str]) – The list of start time in HH:MM:SS format

  • time_stamp_end_1 (List[str]) – The list of end time in HH:MM:SS format.

Returns:

None
