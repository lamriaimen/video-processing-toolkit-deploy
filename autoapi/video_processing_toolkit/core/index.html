video_processing_toolkit.core.get_frame_types(video_file)

Function to extract the type of each frame in a video using ffprobe The frame types are typically: - ‘I’ (Intra-coded): A full image. - ‘P’(Predicted): Stores only the difference from the previous frame. - ‘B’ (Bidirectional): Stores differences using both previous and next frames. :param video_file: Path to the input video file. :type video_file: str

Returns:

A zip object containing tuples of (frame_index, frame_type),

where frame_index is an integer and frame_type is a string (‘I’, ‘P’, ‘B’).

Return type:

zip
